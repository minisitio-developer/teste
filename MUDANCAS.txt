- Rate Limiting (express-rate-limit) — proteção contra brute-force
- Helmet — headers HTTP de segurança
- JWT + RBAC — autenticação com roles (Admin/User)
- bcryptjs — hash de senhas mais seguro
- Validação de input — CPF/CNPJ/Email com validations.js
- Webhook Signature Verification — validação criptográfica
- CORS configurado — controle de acesso cross-origin
- Links com rel=noreferrer — segurança em links externos
2. Melhorias de Performance (⚡)
- Redis Cache — cache de buscas (reduz queries repetitivas ao banco)
- Remoção de database.sync() por requisição — antes rodava sync em toda chamada
- Paginação em buscas e listagens do admin
- Migração CRA → Vite — builds mais rápidos, hot-reload eficiente
- FULLTEXT indexes no MySQL para buscas instantâneas
- Índices compostos (idx_search) para performance de consultas
